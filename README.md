# Daily Blog

A simple client-side daily blogging website built with HTML, CSS and JavaScript. Posts are stored locally in your browser using `localStorage`.

## Structure
- `index.html` – shows today's posts.
- `post.html` – displays a single post.
- `new-post.html` – form for creating new entries.
- `css/styles.css` – styling with glassmorphism effects.
- `js/script.js` – small script managing posts.

All assets are organized in `css/`, `js/` and `images/` folders.
